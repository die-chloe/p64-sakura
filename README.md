# Sakura Wallpaper
for Picotron

My first project in Picotron. I wanted to make a pretty wallpaper limited to the default palette and challenged myself to make it interactive and test Picotrons limits.
Theres probably a lot to still be optimized with storing and sampling of the perlin noise, storing and animating the leaves and grass, but everything fits in the 256k of ROM and uses ~80% of the available cycles for 60fps.
